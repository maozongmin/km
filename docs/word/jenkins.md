# jenkins

```
cmd:
    net start jenkins            // 启动Jenkins服务
    net stop jenkins             // 停止Jenkins服务

linux:
    #jenkins启动
    service jenkins start

    #重启
    service jenkins restart

    #停止
    service jenkins stop
```
